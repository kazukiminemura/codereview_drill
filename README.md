# Code Review Drill

AI開発で必要になるコードレビュー能力を、例題から段階的に学ぶ教材です。

## Web版 Review Gym

ダークIDE形式の演習画面で、章ごとの解説付き例題を読みながら既存の100問に取り組めます。判定、重大度、コード上の事実、発生条件、影響、修正案、テスト案を書いてから回答例と比較します。回答と進捗はブラウザ内へ保存されます。

```bash
python -m http.server 8000
```

起動後に `http://localhost:8000/web/` を開いてください。教材データは既存の `CODE_REVIEW_101.md` と `CODE_REVIEW_101_ANSWERS.md` から読み込むため、問題を更新するとWeb版にも反映されます。

## はじめて取り組む方

- [Code Review 101（解説付き・全100問）](CODE_REVIEW_101.md)
- [Code Review 101 回答ガイド](CODE_REVIEW_101_ANSWERS.md)

各章で2つの解説付き例題を読んでから、論点を一つに絞った10問へ進みます。Web版では各問題の「解説例」タブから、その章の例題をいつでも確認できます。最初から旧ドリルへ進まず、まず101を順番に解いてください。
