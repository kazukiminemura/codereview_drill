# Code Review 101 — 解説から始める100問

1問につき論点は原則1つです。各章は「解説付き例題→10問」の順です。

## 回答の型

事実、発生条件、影響、提案の4点を書きます。「危険です」だけでは根拠不足です。

## 第1章：名前と意図

### 解説付き例題

```python
def calc(a, b): return a * b
```

**解説**：aとbの意味が分からず、単価×数量なのか幅×高さなのか判断できません。名前は処理の意図と単位が伝わるものにします。

**レビュー例**：priceとquantityを受け取る仕様なら calculate_total_price(price, quantity) とする。

### 練習問題

### 問001

```text
def calc(a, b): return a * b
```

引数と関数の目的が分かる名前へ直してください。

### 問002

```

```
